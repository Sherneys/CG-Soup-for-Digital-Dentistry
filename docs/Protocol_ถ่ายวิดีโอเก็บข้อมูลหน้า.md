## ⛔ กฎเหล็กข้อเดียวที่ห้ามพลาด

**"คนนิ่งเหมือนรูปปั้น — กล้องเป็นฝ่ายเคลื่อนรอบหัว"**

SfM (COLMAP) ทำงานโดยสมมติว่า "ฉากแข็ง ไม่ขยับ แล้วกล้องเคลื่อนผ่าน" ถ้ากลับด้าน (กล้องนิ่งแล้วหันหัวเอง) ฉากหลังจะบอกอัลกอริทึมว่ากล้องไม่ขยับ ส่วนหัวที่หมุนจะถูกมองเป็นวัตถุรบกวนแล้ว**ถูกทิ้งทั้งหมด** — ผลคือได้โมเดลของห้อง ไม่ใช่หัวคน

### ตัวอย่างที่พลาด (13 มิ.ย.)

วิดีโอครั้งแรกถ่ายแบบ selfie ถือกล้องนิ่งแล้วหันหัวซ้าย-ขวา + สีหน้า/สายตาขยับระหว่างคลิป → ใช้ไม่ได้เลย เพราะ (1) กล้องไม่เคลื่อน (2) หน้าเป็น non-rigid ระหว่างกวาด

## เช็คลิสต์ก่อนกด record

### คนถูกถ่าย (subject)
- [ ] นั่งบนเก้าอี้ไม่หมุน หลังตรง ศีรษะนิ่งสนิทตลอดคลิป
- [ ] **ตาจ้องจุดเดียวไกล ๆ ไม่กะพริบถ้าไหว** (กะพริบ = ตำแหน่งเปลือกตาเปลี่ยน = non-rigid)
- [ ] สีหน้าค้างท่าเดียวตลอด: คลิป 1 = **neutral ปากปิดสบาย ๆ** / คลิป 2 = **ยิ้มเห็นฟันค้างไว้**
- [ ] เก็บผมออกจากหน้าผากและหู (หูสำคัญต่อ registration) — ใช้ที่คาดผม/hair cap ถ้าจำเป็น
- [ ] ถอดแว่น ถอดต่างหูชิ้นใหญ่

### คนถ่าย (operator — แนะนำให้มีคนช่วย อย่าถ่ายเอง)
- [ ] ใช้**กล้องหลัง** (คมกว่ากล้อง selfie มาก) ความละเอียดสูงสุดที่มี (4K ถ้าได้)
- [ ] **ล็อก exposure + focus**: iPhone = กดค้างบนหน้าจอจนขึ้น "AE/AF LOCK" (กันสี/ความสว่างเพี้ยนข้ามเฟรม)
- [ ] ปิด beauty filter / HDR อัตโนมัติ / โหมดถ่ายกลางคืน
- [ ] ระยะห่าง **50–80 ซม.** ให้หัวเต็มเฟรมพอดี (เผื่อขอบเล็กน้อย)

### สภาพแวดล้อม
- [ ] แสงกระจายสม่ำเสมอ (เปิดไฟห้องหลายดวง / ใกล้หน้าต่างฝั่งไม่ย้อนแสง) — **ห้ามย้อนแสง ห้าม spotlight ข้างเดียว**
- [ ] ฉากหลังมีลวดลาย/รายละเอียดบ้างยิ่งดี (ช่วย SfM จับ feature) — ไม่ต้องเป็นฉากเรียบ
- [ ] ปิดพัดลม/อะไรที่ทำให้ผมปลิว

## วิธีกวาดกล้อง

1. เริ่มจาก**ข้างหูซ้าย** → โค้งผ่านหน้าตรง → จบที่**ข้างหูขวา** (~180°)
2. เคลื่อนช้าและนิ่ง **ใช้เวลา 20–30 วินาทีต่อรอบ** (เร็วกว่านี้ = motion blur = SIFT จับ feature ไม่ได้)
3. ถ้าทำได้ กวาด **3 ระดับความสูง** ในคลิปเดียวหรือแยกคลิป:
   - แถวบน: เฉียงลงจากเหนือระดับตา ~25°
   - แถวกลาง: ระดับตา
   - แถวล่าง: เฉียงขึ้นจากใต้คาง ~15°
4. รักษาระยะห่างจากหัวให้คงที่ตลอดการกวาด (เดินเป็นวงโค้ง ไม่ใช่เส้นตรง)

**ความยาวรวมที่เหมาะ: 30–90 วินาที/คลิป** → สคริปต์จะคัดเฟรมคมให้เหลือ ~70 ภาพเอง (อย่ากังวลว่ายาวไป)

## คลิปที่ต้องถ่าย (อย่างละ 1 ต่ออาสาสมัคร)

| คลิป | สีหน้า | ใช้ทำอะไร |
|---|---|---|
| A1 | neutral ปากปิด | โมเดลหน้า CG-Soup หลัก |
| A2 | **ยิ้มเห็นฟันค้างไว้** | สะพานเชื่อมหน้า ↔ intraoral scan ตอน registration (S5) |

> อาสาสมัครจริงต้องมี consent ตาม PDPA ก่อนถ่ายเสมอ (ข้อมูลหน้า = ระบุตัวตนได้โดยตรง) และเก็บตารางเชื่อม รหัสอาสาสมัคร ↔ ไฟล์ ↔ วันที่

## หลังถ่ายเสร็จ: รัน pipeline

```powershell
$env:PYTHONUTF8 = "1"
cd D:\Project\CG-Soup-for-Digital-Dentistry

# 1. วิดีโอ → เฟรมคม ~70 ภาพ (เลือกเฟรมคมสุดต่อช่วงอัตโนมัติ หนี motion blur)
.venv\Scripts\python.exe src\video_to_frames.py <วิดีโอ.mov> --out data\<ชื่อ>\images --n-frames 70

# 2. SfM → camera poses + dense + coarse mesh
.venv\Scripts\python.exe src\sfm_pipeline.py data\<ชื่อ>\images --workspace output\<ชื่อ>_sfm
# (+ simplify mesh_poisson.ply -> mesh_coarse.ply ด้วย colmap mesh_simplifier ratio ~0.02)

# 3. ภาพย่อสำหรับ train: สร้าง images_4/ (ดูตัวอย่างใน docs/DiffSoup_การใช้งานในโปรเจกต์.md)

# 4. Mask หัว (เช็ค overlay_*.png ทุกครั้งว่าคลุมหัวครบ)
.venv\Scripts\python.exe src\make_masks.py --scene_root output\<ชื่อ>_sfm\dense --mesh output\<ชื่อ>_sfm\mesh_coarse.ply --downscale 4 --radius <ปรับตามหัว>

# 5. Seeds เฉพาะหัว + train
.venv\Scripts\python.exe src\curvature_init.py output\<ชื่อ>_sfm\mesh_coarse.ply --crop-center <x y z> --crop-radius <r> --out output\<ชื่อ>_curv_init.npz
.venv\Scripts\python.exe src\diffsoup_train.py --scene_root output\<ชื่อ>_sfm\dense --init curvature --init_npz output\<ชื่อ>_curv_init.npz --downscale 4 --batch_size 2 --max_faces 5000 --masks output\<ชื่อ>_sfm\dense\masks_4 --out_dir output\diffsoup_<ชื่อ>_curv
```

## สัญญาณว่าวิดีโอใช้ไม่ได้ (เช็คก่อนเสียเวลารัน COLMAP)

| อาการ | สาเหตุ | แก้ |
|---|---|---|
| ฉากหลังนิ่งสนิททุกเฟรมแต่หัวหมุน | ถ่ายแบบหันหัว/เก้าอี้หมุน | ถ่ายใหม่ — กล้องต้องเคลื่อน |
| sharpness ที่สคริปต์รายงานต่ำ (<100) เกือบทุกเฟรม | กวาดเร็วไป/มือสั่น/แสงน้อย | กวาดช้าลง เพิ่มแสง |
| สีหน้า/เปลือกตาต่างกันระหว่างเฟรมต้น-ท้าย | ขยับหน้า/กะพริบบ่อย | ถ่ายใหม่ ค้างสีหน้า |
| สีผิวเพี้ยนข้ามเฟรม | ไม่ได้ล็อก exposure | ล็อก AE/AF แล้วถ่ายใหม่ |
